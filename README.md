# ABOUT

Android keyboard mapping to add "ABNT2"like functionalities to the US layout.

I hijacked the `RIGHT ALT`, `Tilde (~)`, `Grave Accent` and `Circumflex Accent (^)` keys. So you need to be fine with that.
